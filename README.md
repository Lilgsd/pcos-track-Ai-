# pcos-track-Ai-
Apa itu PCOS-Track? PCOS-Track adalah aplikasi berbasis kecerdasan buatan (AI) yang dirancang untuk membantu wanita melakukan skrining awal terhadap risiko Polycystic Ovary Syndrome (PCOS). Dengan memanfaatkan algoritma Machine Learning, aplikasi ini menganalisis berbagai gejala fisik dan indikator klinis dasar kemungkinan risiko PCOS
